### 🔨 [OSForge](https://github.com/plocemourasouza/osforge) — MIT Licensed

> OSForge é um framework open-source de skills para agentes de IA, projetado para elevar a produtividade de desenvolvedores que utilizam Claude Code e Cursor. Com 31 skills especializadas, 7 agentes pré-configurados, 4 regras always-on e hooks Python prontos para produção, o projeto resolve um problema crítico dos frameworks monolíticos tradicionais ao introduzir carregamento on-demand e um sistema de roteamento em 3 camadas — reduzindo o consumo base de tokens em ~83%, mantendo contexto cirúrgico, segurança integrada contra injeção de código e total compatibilidade com o stack moderno (Next.js, TypeScript, Prisma, Supabase).

- 🧩 **31 agent skills** | **7 agents** | **4 always-on rules**
- 🐍 Python hooks para **Claude Code** e **Cursor**
- ⚡ Sistema de **model routing em 3 camadas**
- 📉 Redução de **~83%** no consumo base de tokens vs. frameworks monolíticos
- 🔒 Hooks de segurança integrados: detecção de credenciais, prevenção de injeção, validação Zod
- 📜 **v3.1** — Carregamento de skills on-demand, pronto para produção
